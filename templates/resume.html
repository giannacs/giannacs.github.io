Notes on Multivariate Linear Regression

One of the biggest challenges in an analytical field is converting a real-world problem into a mathematical one. This can be achieved with a mathematical parametric model. To extract information about the model one can use the process of linear regression. It is especially relevant in the field of statistics, machine learning, etc. The aim of this project is to illustrate the power of linear regression.

  • Linear regression for pedestrians
  • With some linear algebra, get a powerful estimating tool
